# Instagram

## Requirements

- Python (>3.6)
- Secrets JSON File

## Installation

```
pip install -r requirements.txt
```

## Secrets JSON File

```
/secrets.json
{
  "AWS_ACCESS_KEY_ID": "<AWS AccessKeyID (S3 permission)>",
  "AWS_SECRET_ACCESS_KEY": "<AWS SecretAccessKey (S3 permission)>"
}
```

## Run development server

```
python manage.py runserver
```
